# LTI 1.3 JWT

Library with JWT implementations of LTI's JWT interfaces

## Usage

Just add the dependency to your `pom.xml` file:

```xml
  <dependency>
    <groupId>edu.uoc.elc.lti</groupId>
    <artifactId>lti-1.3-jwt</artifactId>
    <version>0.0.2</version>
  </dependency>
```
